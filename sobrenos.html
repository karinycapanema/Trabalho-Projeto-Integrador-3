Este site foi criado apartir de um trabalho de Projeto Integrador, unindo as disciplinas de Desenvolvimento Web e Biologia para compartilhar informações confiáveis sobre alimentação saudável. Nosso objetivo é mostrar como pequenas escolhas podem melhorar a saúde e o bem-estar. O projeto foi desenvolvido por Eloisa, Júlia B, Júlia C, Kariny e Pietra, aluna do IFC Araquari que juntaram ideias para criar um espaço de aprendizado e incentivo a hábitos alimentares equilibrados.
